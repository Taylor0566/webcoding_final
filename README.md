# 成绩管理教学平台 (Grade Management System)

## 项目简介
本项目是一个基于 Web 的成绩管理教学平台，旨在提供高效、便捷、专业的教学教务管理服务。系统涵盖了游客、学生、教师、教学管理员和系统管理员五类用户角色，实现了课程管理、选课修读、成绩录入与查询、用户管理等核心功能。

## 功能特性

### 1. 公共模块
*   **首页展示**：简洁现代的界面设计，展示热门课程信息。
*   **课程搜索**：支持按课程名称、ID 等关键字实时搜索公开课程。
*   **安全登录**：
    *   支持多角色账号登录。
    *   **安全机制**：采用加盐哈希模拟密码存储，具备连续尝试失败锁定账号（3次错误锁定5分钟）的安全策略。

### 2. 学生端 (Student)
*   **课程修读**：
    *   **选课中心**：浏览所有已发布课程并进行选修操作。
    *   **我的课程**：查看当前正在修读的课程及学习状态。
*   **成绩查询**：
    *   **成绩单**：查看所有修读课程的详细成绩，包括平时分、期中、期末及总评成绩。

### 3. 教师端 (Teacher)
*   **课程管理**：
    *   **发布课程**：创建新课程，设置学分、上课时间、地点及课程简介。
    *   **编辑课程**：维护已有课程的详细信息。
*   **成绩管理**：
    *   **成绩录入**：按班级名单录入学生成绩（平时、期中、期末）。
    *   **自动计算**：系统根据预设比例（如 30:30:40）自动计算总评成绩。

### 4. 教学管理员 (Edu Admin)
*   **基础数据管理**：学生信息的批量导入（模拟）、删除等维护操作。
*   **教务管理**：排课管理、教室安排等功能入口。

### 5. 系统管理员 (Sys Admin)
*   **审计日志**：查看系统中关键操作的操作日志（如用户登录、数据变更）。
*   **数据维护**：系统数据的备份与恢复功能。

## 技术栈
*   **前端核心**：HTML5, CSS3, JavaScript (ES6+)
*   **样式设计**：采用现代极简主义设计语言，使用原生 CSS 变量实现响应式布局与交互动画。
*   **数据存储**：使用 `localStorage` 模拟后端数据库，实现数据的持久化存储与状态管理。
*   **无后端依赖**：纯前端架构，开箱即用，无需配置服务器环境。

## 快速开始

1.  **获取代码**：下载或克隆本项目到本地。
2.  **运行项目**：直接双击根目录下的 `index.html` 文件，使用现代浏览器（Chrome, Edge, Firefox, Safari 等）打开即可。
3.  **体验功能**：使用下方提供的测试账号登录体验各角色功能。

## 注意事项
*   本项目为前端演示系统，所有数据存储在浏览器本地（LocalStorage），清除浏览器缓存会重置数据。
*   系统中的“文件导入/上传”均为模拟交互，不会真实上传文件到服务器。
